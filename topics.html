GeoScripting project week

topics

During the 4th week of the GeoScripting course each group will be required to carry out a small geo-scripting project. The specific objectives and assessment criteria of these project will be presented to you by Jan later. You can decide which topic you want to work on by either coming up with your own topic idea (which will need to be discussed and approved by one of the course lecturers), or selecting a topic from the list of topics we've compiled.

This year's topic are the following

Interactive tool for visualization of change detection results

We have algorithms in house designed to detect change on satellite images time-series. The objective of this project would be to design an interactive tool (with shiny or another tool) that allows the user to interactively produce and visualize maps of land cover change.

Tools

  • R
  • Shiny
  • bfastSpatial
  • Leaflet

Data:

  • MODIS or Landsat

Planet Labs topics

Planet Labs is an Earth Imaging company that launches micro-satellites. The satellites capture images of the earth at high spatial resolution (3-5m) and, because of their high number, with a high temporal frequency. While users would normally need to pay for accessing the data, Planet Labs has made its data archive available for the state of California for users and developpers like you to develop and test methods and applications. We defined a list of topics using Planet Labs data that we think would present interesting challenges for the project week of the geoScripting course.

Urban tree detection from Very High Resolution data

The objective of this topic is to develop a spatial segmentation method using scripting with open source tools to automatically identify urban trees from very high spatial resolution imagery

Tools

  • python
  • Planet Labs API
  • Orfeo Toolbox
  • python gdal bindings or rasterio
  • scikit-image

Datasets

  • Planet Labs data
  • Urban forest map for validation

Crop type mapping using temporal dynamics

For an agricultural area of California you'll develop a method using the Planet Labs API and the geo-scripting knowledge you have acquired during the past 3 weeks to differentiate crop types. Planet Labs data have limited spectral characteristics but high temporal resolution.

Tools

  • python
  • Planet Labs API
  • python gdal bindings or rasterio
